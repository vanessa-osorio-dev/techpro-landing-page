# 🎧 TechPRO - Responsive Landing Page

## 📋 Descripción
 
Landing page responsive desarrollada con HTML, CSS y JavaScript, enfocada en buenas prácticas de maquetación web, optimización de imágenes y diseño moderno.
 
## ✨ Características
 
- **Diseño Responsive**: Adaptado para todos los dispositivos (móvil, tablet, desktop)
- **Optimización de Imágenes**: Soporte para múltiples formatos (AVIF, WebP, PNG, JPG, SVG)
- **Detección Automática de Formatos**: Script JavaScript que detecta los formatos de imagen soportados por el navegador
- **Diseño Moderno**: Uso de gradientes, animaciones y transiciones CSS
- **Variables CSS**: Sistema de diseño con variables personalizadas para consistencia
- **Tipografía Google Fonts**: Integración de fuentes Roboto y Lato
- **Grid y Flexbox**: Layout moderno con CSS Grid y Flexbox

 
## 🚀 Demo en Vivo

## 🛠️ Tecnologías Utilizadas
 
- **HTML5**: Estructura semántica
- **CSS3**: Estilos con variables, grid, flexbox y animaciones
- **JavaScript (Vanilla)**: Detección de formatos de imagen
- **Normalize.css**: Reset de estilos cross-browser
- **Google Fonts**: Roboto y Lato

## 🌐 Optimización de Imágenes

El proyecto implementa una estrategia de optimización para mejorar el rendimiento y la compatibilidad entre navegadores.

- **AVIF y WebP:** formatos modernos para reducir el tamaño de las imágenes.
- **Fallback automático:** uso de PNG y JPG cuando el navegador no soporta formatos modernos.
- **Detección con JavaScript:** el archivo `imagenes.js` identifica los formatos compatibles y aplica automáticamente la mejor opción.


## 📱 Responsive Breakpoints
- Móvil: < 768px
- Tablet: 768px - 992px
- Desktop: > 992px
- Large Desktop: > 1600px
